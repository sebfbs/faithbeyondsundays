# Build Log — 2026-05-01 (Session 5)

## Dark Mode Polish — Remaining Issues Fixed

### What was broken
After session 4 wired the dark mode toggle, several screens still looked wrong:
- **Home screen** — sky gradient was still showing (the `getSkyGradient()` call in Index.tsx doesn't read CSS variables, it's plain JS)
- **Daily Spark overlay** — same issue, used `getSkyGradient()` directly
- **Home screen cards** — all cards used `hsl(0 0% 100% / 0.8)` frosted glass designed to float over the sky gradient. In dark mode = grey blobs.
- **Community Activity card** — same frosted glass issue (CommunityPulse.tsx)
- **Journal Suggested Scripture block** — hardcoded `hsl(210 55% 94%)` light blue background + dark blue text
- **Blue pills ("Daily Reflection" tag)** — hardcoded dark blue text on light blue background, no dark override

### What was fixed

#### Index.tsx
- Added `useTheme` from `next-themes`, read `resolvedTheme`
- Homepage `<main>` background skips `getSkyGradient()` when `resolvedTheme === "dark"` — falls through to `hsl(var(--background))`
- Status bar overlay area also skips `getSkyGradientTopColor()` in dark mode

#### DailySparkOverlay.tsx
- Added `useTheme`, checks `resolvedTheme === "dark"`
- Dark mode: deep navy gradient (`hsl(225, 45%, 8%)` → `hsl(215, 35%, 20%)`) instead of sky gradient
- Light mode: unchanged sky gradient

#### HomeTab.tsx
- All 7 card instances that used `background: "hsl(0 0% 100% / 0.8)", backdropFilter: "blur(20px)"` replaced with `bg-card`
- Applies to: Spark card, Reflection card, their loading skeletons, empty state, Connect to Church button, Quick Links buttons, Reflection overlay modal
- Light mode: white cards over sky gradient (same as before — `bg-card` = white in light mode)
- Dark mode: dark navy cards matching every other screen in the app

#### CommunityPulse.tsx
- Same frosted glass → `bg-card` fix
- Avatar overlap separator: was `border-2 border-white` (white ring visible in dark mode) → changed to `boxShadow: "0 0 0 2px hsl(var(--card))"` — invisible against card but still separates overlapping circles

#### JournalTab.tsx
- Suggested Scripture block: replaced hardcoded `hsl(210 55% 94%)` background and `hsl(207 65% 40%)` text with `colors.accentBg` and `colors.pillTextSoft` from `useAccentColors()` — already dark-mode aware

#### index.css
- Added `.dark .blue-pill { background: hsl(215 40% 22%); color: hsl(210 65% 70%); }` — deep blue bg + light blue text in dark mode
